# 🐍 贪吃蛇游戏 (Snake Game)

经典的贪吃蛇游戏，使用 Python 和 pygame 实现。

## 📸 游戏截图

![Game Preview](preview.png)

## ✨ 特性

- 🎮 经典贪吃蛇玩法
- 🎨 简洁清爽的界面设计
- ⌨️ 支持方向键和 WASD 控制
- 📊 实时分数显示
- 🔄 游戏结束后可重新开始

## 🚀 快速开始

### 环境要求

- Python 3.6+
- pygame

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行游戏

```bash
python snake.py
```

## 🎮 控制方式

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| SPACE | 重新开始（游戏结束后） |
| ESC | 退出游戏 |

## 📋 游戏规则

1. 控制蛇吃掉红色食物
2. 每吃一个食物，蛇身变长，分数 +10
3. 撞到墙壁或自己，游戏结束
4. 尽可能获得高分！
