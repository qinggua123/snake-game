# 🐍 贪吃蛇游戏 (Snake Game)

经典的贪吃蛇游戏，支持 Python 桌面版和 HTML5 网页版！

## ✨ 功能特性

### 🎨 6 种精美皮肤
- **经典绿** - 复古经典配色
- **蓝色风暴** - 清爽蓝色主题
- **紫色幻影** - 神秘紫色风格
- **黄金传说** - 奢华金色体验
- **黑客帝国** - 极客绿色代码风
- **暗黑模式** - 护眼深色主题

### 🏆 最高分记录
- 每个主题独立记录最高分
- 自动保存到本地文件/浏览器
- 新纪录会有特殊提示！

### 🔊 音效支持
- 吃食物音效
- 游戏结束音效
- 可随时开关音效
- 无需外部文件（合成音效）

### 📱 多平台支持
- **Python 桌面版** - 完整功能体验
- **HTML5 网页版** - 即开即玩，支持手机

---

## 🚀 快速开始

### 方式 1：Python 桌面版

#### 安装依赖
```bash
pip install -r requirements.txt
```

#### 运行游戏
```bash
python snake.py
```

#### 控制方式
| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| T | 切换主题 |
| M | 开关音效 |
| P | 暂停 |
| SPACE | 重新开始（游戏结束后） |
| ESC | 退出游戏 |

---

### 方式 2：HTML5 网页版

直接在浏览器打开 `index.html`，或者：

```bash
# 使用 Python 内置服务器
python -m http.server 8000

# 然后访问 http://localhost:8000
```

**手机也能玩！** 网页版支持触摸控制。

---

## 📋 游戏规则

1. 控制蛇吃掉红色食物
2. 每吃一个食物，蛇身变长，分数 +10
3. 撞到墙壁或自己，游戏结束
4. 尽可能获得高分！

## 🏆 最高分记录

游戏会自动保存每个主题的最高分到 `highscores.json`：

```json
{
  "classic": 150,
  "blue": 120,
  "purple": 90,
  "golden": 200,
  "matrix": 180,
  "dark": 110
}
```

网页版使用浏览器 LocalStorage 保存。
